Lately, I am converting thousands of technical documents into DITA, an XML data model.

The project uses the Ruby on Rails environment. Ruby supports a really cool HTML and XML parser, Nokogiri.

Nokogiri and I like to make magic:

  • Converting text and HTML into perfectly structured DITA topics,

  • Reproducing customized DITA topics from the command line,

  • Customizing batches of DITA topics with a few keystrokes, and

  • Performing amazing XSLT transformations with Ruby models.
