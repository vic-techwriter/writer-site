I manage technical documentation in the global semiconductor industry.

Lately, I've been converting thousands of technical documents into DITA, an XML data model.

To convert the content, I'm using the Ruby on Rails environment. Ruby contains a really cool HTML and XML parser, Nokogiri.

Nokogiri and I make magic:

  • Converting text and HTML into perfectly structured DITA topics,

  • Reproducing customized DITA topics from the command line,

  • Customizing batches of DITA topics with a few keystrokes, and

  • Performing amazing XSLT transformations with Ruby models.
